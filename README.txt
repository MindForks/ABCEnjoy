Команда MINDFORKS

Сервис ABCEnjoy это поисковая система для всех видов развлечений.

Он также помогает предпринимателям, организаторам различных развлечений привлечь клиентов.

Решает боль пользователей в быстром поиске всех доступных видов развлечений в заданной локации.

Решает боль провайдеров услуг в презентации и продвижении развлечений.

Это Booking.com только вместо номеров в отелях пользователь выбирает, где он будет проводить свое свободное время. 

Сервис будет собирать и размещать все доступные развлечения, в которых нет оплаты, например :
день города, дни иностранных государств, фестивали, концерты уличных музыкантов, семинары, благотворительные мероприятия и много другого.

Это социальная миссия проекта, которая будет помогать привлекать много пользователей.

Алгоритм работы сервиса

Базовая версия продукта позволяет:

1.	Провайдерам, заинтересованным в привлечении клиентов, размещать информацию о развлечении на сайте.
2.	После модерации (в будущем автоматизированной) эта информация появляется в общей базе.
3.	Пользователь в поисках вариантов для проведения досуга заходит в приложение.
4.	Он фильтрует информацию исходя из даты, временного промежутка, цены, видов развлечений и т.д.
5.	Полученную отфильтрованную информацию можно отсортировать по близости к потенциальному покупателю (геолокация), стоимости, релевантности (исходя из личных предпочтений в истории браузера), отзывам.
6.	После этого пользователь выбирает развлечение и оплачивает его через платежную систему или получает доступ к мероприятиям которые не требуют оплаты. 
7.	Только после оплаты можно будет оставить отзыв. По сравнению с TripAdvisor, который могут используют для чёрного PR конкуренты.
